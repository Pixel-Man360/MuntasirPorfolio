My Contributions

  • I worked almost solo on this project. My main task was to collaborate with the artist for all the 2D assets and my senior for all the 3D assets, game design, game feel etc and work on the whole project.
  • I implemented:
  • Gameplay system. Swipe Up, Down, Left, Right to move the cute rabbit from one cell to another. Collect resources as necessary, fight enemies, find the correct path and complete the game within limited moves.
  • Core Gameplay loop. Both win and lose conditions. Go to the finish line until the moves run out to win. If no moves left or killed by an enemy you lose.
  • Camera System. Basic Camera follow and zoom in-out.
  • Enemy AI. Melee and shooter types. (Basic FSM).
  • Inventory for gems and collected resources per level.
  • UI/UX Systems. Collaborated with the artist and senior to properly implement the UI visually along with all the tween animations.
  • Editor Tool to create levels. I made a grid-based level editor to specify grid size and each block type. The levels were saved as unity scriptable objects and loaded to play the game in a single scene.
  • Audio system.
  • Tutorials system.
  • Save load system.
  • I designed:
  • Some of the VFX using the particle system
  • Some basic shaders like damage flash, fog for a better game feel.
  • All the different levels (20 total)
