  • I implemented:
    • Core Gameplay Systems:-
      • Player Behavior-
        • Joystick based semi top-down all directional movement system.
        • Player can interact with the world using different triggers and colliders.
        • Player auto targetting and shooting was implemented using a sphere collider trigger for performance and simplicity.
        • There was an internal capsule collider for the player which was used for interactions with the turret unlock or upgrade zones, attacks zones and enemies.
        • The in game resource for unlocking and upgrading the turrets was radiation token (Green). I made a system where the current area will start with a fixed amount of tokens and enemies would drop a certain amount of radiation tokens.
        • The collection and dropping of the tokens by the player were implemented with a custom bezier curve system for smooth animations and performance (Used coroutines).
