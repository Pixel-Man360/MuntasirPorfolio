Fight for World

Big Bang Studio Ltd.

Genre: Action, Tower Defence, Top-Down shooter, Hybrid-Casual Platform: Android Role: Game Developer Technology Used: Unity, C# Tested with: Homa Short Description: An action packed tower defence game where the players have to defend the payload while attack the enemy base in order to clear each area, destroy the mother base and detoxify each world.     âœ… How I Achieved It

    • Core Gameplay Systems:-
      • Base Player Behavior-
        • Implemented Joystick based semi top-down all directional movement system.
        • Player can interact with the world using different triggers and colliders.
        • The collection and dropping of the tokens by the player were implemented with a custom bezier curve system for smooth animations and performance (Used coroutines).
      • Toxic area system-
        • Each area of a world is separated by a safe zone and a toxic zone. Safe zones are where players can place turrets and heal themselves. The first area is always safe.
        • We can start the fight when the player enters the fight area and the wave starts.
        • Implemented a system where player takes damage over time when entering a toxic area. Players will get an 8 seconds force field for safety. After that they will take damage quickly.
        • Implemented a generator system as a temporary safe zone that players can activate using Radiation Tokens to heal themselves and stop the toxic damage.
      • Champion System-
        • Implemented a champion system where players can unlock and upgrade unique champions to play with.
        • Players can choose from a pool of 4 champions to play with.
        • Each champion are available after finishing a world and/or area. After that players need to unlock them with Gems or Ads. The first champion Master Rick is unlocked by default.
        • Each champion has distinct abilities and playstyles, encouraging diverse strategies.
        • Players can level up their champions through player fragments and coins gained from battle rewards, chests and shop.
        • Each champion has a special ability called Frenzy mode which triggers after killing a certain number of enemies. During frenzy mode each champion uses their own unique special skills. I didn't implement their behaviors but I added the VFX and SFX for champion abilities to enhance gameplay immersion
      • Gun System-
        • Implemented a versatile gun system that supports multiple weapon types.
        • Each weapon has unique stats such as damage, fire rate, and reload time.
        • Each weapon unlocks after finishing a certain world and/or area (The first weapon is unlocked by default).
        • Players can pick a weapon out of 6 different weapons from the weapons tab in the main menu.
        • Designed a weapon upgrade and rank up system that allows players to enhance their weapons' stats.
        • There are 4 different ranks for each weapon which are common (Blue), Rare (Green), Epic (Orange), Legendary (Red), each increasing the level cap.
        • Players can upgrade weapons using coins and rank up weapons using weapon fragments gained from battle rewards, chests and shop.
        • Added a skin system for 3 specific guns. All of them had a unique skin which would unlock from daily login or can be bought.
        • Each skin has its own VFX and animations.
        • Added VFX and SFX for upgrading and rank up for each gun and also added UI and VFX to show each rank.
        • I didn't implement the shooting system but I added the VFX and SFX for shooting, hitting and damage number and also designed different projectiles and audio for different guns.
      • Turret System-
        • Implemented a turret system that allows players to deploy stationary turrets at fixed positions in battle.
        • Designed unique behaviors for 7 turrets out of 12.
        • Added unique VFX and SFX for projectiles or abilities for all turrets.
        • Engineered a deck system for turrets where players can pick 6 turrets out of 12 available options on the turret section of the player tab in the main menu.
        • When a player drops enough radiation token (Resource to unlock turrets) inside an unlock area, a popup is shown to pick 2 different turrets from the deck pool. The first is free while the second has a cost.
        • Players can refresh the options for free or with ads.
        • Picking an option drops the selected turret into the area.
        • Players can upgrade the turret in game to increase a turret's ability and visuals. They can also upgrade the base stats of a turret using coins and specific turret fragments found from battle rewards, chests and shop.
        • Used unity's scriptable objects to manage the base and upgradable stats of each turret.
      • Skill Tree System-
        • The linear skill tree system was made to boost the player base/payload. I was in charge of updating the existing UI and VFX and integrating the system into gameplay.
      • Tutorial System-
        • Implemented a simple gameplay tutorial system to guide players through the basics of each mechanics or what to do during gameplay.
        • Also Implemented a main menu tutorial system by expanding the existing one to guide players into using all the main menu features such as gacha equipment equipping, champion upgradding, gun equip and upgrade and so on.
      • Currency System-
        • Created a modular currency system that allows players to earn and spend in-game currency.
        • There were a total of 6 different types of currency and one gameplay resource.
        • Each currency type has its own unique use and can be earned through various gameplay activities.
        • The currencies include:
          • Gold. Used everywhere.
          • Gems. Used to buy chests and unlock champions.
          • Player Fragments. Used to upgrade champions.
          • Turret Fragments. Used to upgrade turrets. Each turret has a unique fragment type can be used on that specific turret only.
          • Weapon Fragments. Used to upgrade weapon rank.
          • Radiation Token. This is a gameplay resource collected by killing enemies and used to unlock turrets and generators and upgrade turrets during gameplay.
      • Gacha / Equipment System-
        • Implemented a gacha system for acquiring new equipment for the champions
        • There are a total of 6 different equipments, each with its own unique abilities.
        • Each equipment has 4 different ranks similar to the guns. Each rank increases the equipment's stats.
        • Added a system to craft a higher ranked equipment by merging 3 same ranked and type equipments. Players can also equip or unequip any one of the equipment in their specified slots.
        • Added vfx and sfx for crafting and equipping equipments.
        • Used Unity's scriptable objects to define stats and abilities for each equipment type and rank.
        • The 6 equipments are:
          • Dog Tag. Increases critical chance and for non-common ranks increases fire range.
          • Knife. Increases attack damage and for non-common ranks increases attack of specific weapons.
          • Bandage. Increases self-heal amount and for non-common ranks increases self-heal amount of payload/player base.
          • Gas mask. Increases forcefield duration of the generators and for non-common ranks decreases toxic damage taken.
          • Vest. Decreases enemy damage taken and for non-common ranks decreases toxic damage taken.
          • Boots. Increases movement speed and for non-common ranks creates an attack dodge chance where the player gets no damage.
      • Powerups System-
        • Created a powerup system that gives an extra boost to the players when fighting. Each has their own cooldown.
        • Used unity's scriptable objects to define the powerups and their effects.
        • Implemented 4 out of 6 powerup behaviors.
        • Similar to the turret deck system I created a system where players can pick at most 2 out of 6 powerups. The option was added inside the player tab in the main menu.
        • Added vfx and sfx for the selections and some of the powerup activations such as Ice grenade and mines etc.
      • Misc-
        • Added an indicator system on the ground which shows the distance and direction of the attack area, enemy bases and generators when necessary.
        • The player base movement was implemented using Unity Spline. Some of the level designs were also done using Spline.
        • Implemented a world end cinematic where the payload/player base moves towards the mother base of the enemy, opens hatchet to reveal turret, charges up and fires. Then camera pans to the mother base, it gets destroyed and the end result screen pops up after some sfx.
        • Implemented end result screens of both win and lost and also the reward system for both.
        • Implemented a centralized reward system along with an internal APIwhich is used in many parts of the game. Mostly on the main menu where different rewards are given.
    • UI Systems:-
      • ðŸš€ I designed and implemented most of the UI systems in the game. I used the GUI pro survival pack provided by the studio for this one. This is my best UI work and I'm proud of it.
      • The reset and time system for every single reward is made using C# DateTime. There is a calculation for week passed as well.

      • Main menu Shop-
        • Implemented a basic main menu shop on the shop tab to buy rewards, claim free rewards or watch ads to get rewards. Everything resets after 24 hrs.
      • Missions-
        • Implemented a mission system where players can complete tasks for rewards.
        • Created a user-friendly interface for the mission tab with clear objectives and rewards.
        • Used Unity's scriptable objects to define daily and weekly missions and all mission complete rewards.
      • Chest-
        • Implemented instant chests inside shops and also battle reward chests that opens after a certain amount of time.
        • Created the entire chest opening and unlock animation sequences using Dotween.
        • Added vfx and sfx for every effect.
      • Feature Unlocked System-
        • Created a modular system to showcase when a new main menu feature such as player Upgrade, weapon upgrade, shop and so on has unlocked.
        • Made this system fully modular so that it can be used in other projects as well. In fact this feature was used in 4 different project after this one.
      • Daily loging reward-
        • Used unity's scriptable systems to create a pool of 4 weeks worth of data. Each data gets reset after 7 days and each reward item is claimable after logging in the next day.
      • 7 day win rewards-
        • Created a 7-day win reward system where players can earn increasing rewards for consecutive wins (5 win max).
      • Offline rewards-
        • Implemented an offline reward system where players can earn rewards for being offline for a certain period of time.
      • Player Profile-
        • Created a player profile system where players can view their progress, change avatars and their user name.
      • Main Menu design-
        • Designed the entire main menu layout and navigation for a user-friendly experience.
        • Separated 5 different views with 5 different tabs in the bottom part.
        • The battle view shows the current world name and background and also the selected champion.
    • Level Design:-
      • Designed areas for world 1, 3, 5 and 6 out of 8. Each world has around 5 - 7 areas.
    • Monetization Systems:-
      • Ads-
        • Implemented and placed rewarded ads using the Homa SDK in various parts of the game to enhance player engagement and retention.
