  • I implemented:
    • Core Gameplay Systems:-
      • Pin Gameplay-
        • Tap to unscrew a pin.
        • Tap on an empty slot to screw a pin.
        • Pin will only be screwed if the slot has no overlap.
      • Card Gameplay-
        • Unscrewed cards will fall down and moved to a card slot on top.
        • The cards are sorted by colors and elements. Each card will have a different color and sometimes a color and an element.
        • These cards are needed to be placed onto the box on top of the gameplay. Each box has a specific color and three empty slots inside. The cards are needed to be placed my color and element inside the slots.
        • If the color of the card doesn't match the box, it will go on to the temporary slot.
        • If all the temporary slots are filled, that level is lost.
        • If a box is filled, another box of a different color will come forward. If the temporary slots have any card of the same color and/or element, it gets filled immediately.
        • The cards will be sorted by color into the temporary card slots.
        • This gameplay system was created using Unity Hinge joint. I connected the cards and pins using hinge joints to allow for smooth movement and interaction. Turning of a joint makes the card rotate and fall.
    • Powerup systems:-
      • Extra pin slot powerup-
        • Clicking this powerup gave an extra pin slot to the players.
        • This was done by having a predefined hidden pin which was activated when the powerup was clicked.
      • Extra card slot powerup-
        • Clicking this powerup gave an extra card slot to the players.
        • This was done by having a predefined hidden card slot which was activated when the powerup was clicked.
      • Hammer powerup-
        • Clicking this powerup makes the pins scale up and down. When pin is clicked, a hammer spawns and hits the pin and it disappears.
  • I designed:
    • Editor Tools:-
      • Level Design tool-
        • The level design was a repeatative process. To save time I made 2 different editor tools.
        • The first tool was responsible for setting up the hinge joints, rigidbodies and colliders for each card and pin. This made connecting a card to a pin or another card easier.
        • The second tool was responsible for setting up the visual layers for each of the 2d cards (as some cards had to stay on top of others).
        • I used Unity's scriptable object system for the level data such as number of briefcase, what type and color of cards each briefcase slot contained and whether or not it's a harder level.
    • Main Menu:-
      • Level scrolling-
        • The level scroller was created using sliders and some calculations to determine the spacing between each level.
        • Hard levels had different graphics to indicate their difficulty.
      • Spin Wheel-
        • The spin wheel was created to give players some powerups or coins as reward.
        • The wheel was created using Unity's UI system. The spinning effect was done using dotween. I divided the wheel into 8 sections and each section had a different reward.
        • When the spin button is pressed, a section is randomly chosen. Then I calculated the number of rotation required to get to a slot by using 360 / slotCount formula. Then I just added some offset for the spin count and that acted as the duration.
        • The spinning ease out was done using dotween.
        • I modified the sfx for the spin using Audacity and made it as long as the spin duration.
